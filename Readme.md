# headless-blog
A headless blog created in spring boot. Integrated it with any frontend tech of your choice and start blogging.

## Features

- [ ] Authentication
  - [ ] Admin login
  - [ ] User login
- [ ] Posts
  - [ ] Get all posts
  - [ ] Save post
  - [ ] Update post
  - [ ] Delete post
- [ ] Categories
  - [ ] Get all categories
  - [ ] Save category
  - [ ] Update category
  - [ ] Delete category
- [ ] Tags
  - [ ] Get all tags
  - [ ] Save tag
  - [ ] Update tag
  - [ ] Delete tag
- [ ] Search
  - Search post by title
  - Full text search of post (title, description)
  - Get posts by category name
  - Get post by tag
